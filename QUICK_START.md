## ✅ Wymagania (sprawdź przed startem)

Musisz mieć zainstalowany **Node.js**:

```bash
node --version
npm --version
```

Jeśli brakuje - pobierz z: https://nodejs.org/

---

## 🚀 4 Kroki do Uruchomienia

### Krok 1: Otwórz PowerShell/Terminal

Otwórz PowerShell (Windows) lub Terminal (Mac/Linux)

### Krok 2: Przejdź do folderu

```bash
cd "c:\Users\marec\OneDrive\Desktop\Phishing - Marek Piłat\phishing-education-app"
```

Lub na Mac:
```bash
cd ~/Desktop/"Phishing - Marek Piłat"/phishing-education-app
```

### Krok 3: Zainstaluj zależności

```bash
npm install
```

⏳ Czekaj 2-3 minuty...

### Krok 4: Uruchom aplikację

```bash
npm start
```

✨ **GOTOWE!** Aplikacja powinna się otworzyć w przeglądarce.

---

## 🌐 Gdzie powinna się otworzyć?

```
http://localhost:3000
```

Jeśli nie otworzyła się automatycznie, wklej to do przeglądarki.

## 🎮 Co robić, jeśli coś pójdzie nie tak?

### ❌ "npm nie jest rozpoznane"
- Zainstaluj Node.js z https://nodejs.org/
- Uruchom ponownie

### ❌ "Port 3000 jest już w użyciu"
```bash
$env:PORT=3001; npm start
```

### ❌ "npm install się wiesza"
```bash
npm cache clean --force
npm install
```

### ❌ "Aplikacja się nie otwiera"
Wpisz ręcznie w przeglądarce: `http://localhost:3000`
